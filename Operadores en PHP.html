Operadores Matemáticos

Operador Operación Ejemplo Resultado
+ Suma $x + $y (php echo $x+$y;) Suma dos números
- Resta $x - $y (php echo $x-$y;) Resta dos números
* Multiplicación $x * $y (php echo $x*$y;) Multiplica dos números
/ División $x / $y (php echo $x/$y;) Divide dos números
% Módulo $x % $y (php echo $x%$y;) Obtiene el residuo de una división
** Elevación $x ** $y (php echo $x**$y;) Eleva a 'x' potencia un número

Operadores Asignación

Operador Operación Ejemplo Descripción
= Guardar / Asignar $x = $y (php echo $x=$y;) Guarda el valor de la variable del lado derecho en la variable del lado izquierdo
+= Guardar y Sumar $x += $y (php echo $x+=$y;) Suma y Guarda el resultado de una suma
-= Guardar y Restar $x -= $y (php echo $x-=$y;) Resta y Guarda el resultado de una suma
*= Guardar y Multiplicar $x *= $y (php echo $x*=$y;) Multiplica y Guarda el resultado de una suma
/= Guardar y Dividir $x /= $y (php echo $x/=$y;) Divide y Guarda el resultado de una suma

Operadores de Comparación

Operador Operación Ejemplo Descripción
== Igualdad $x == $y (php echo $x==$y;) Compara dos números entre si
=== Idéntico $x === $y (php echo $x===$y;) Compara dos números iguales
!= Diferente $x != $y (php echo $x!=$y;)
*= Guardar y Multiplicar $x *= $y (php echo $x*=$y;) Multiplica y Guarda el resultado de una suma
/= Guardar y Dividir $x /= $y (php echo $x/=$y;) Divide y Guarda el resultado de una suma

Operadores Lógicos

Operador Operación Ejemplo Descripción
&& Y $x && $y (php echo $x&&$y;) Devuelve un valor si ambos operadores son ciertos
|| O inclusivo $x || $y (php echo $x||$y;) Devuelve un valor si uno de los operandos es cierto
xor O exclusivo $x xor $y (php echo $xxor$y;) Realiza una exclusión lógica en dos expresiones
! No ! $x (php echo !$x;) Invierte el valor de cualquier otro operador
